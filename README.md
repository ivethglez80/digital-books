## Tecnologías
- **Frontend:**
  - HTML, CSS, JavaScript
  - Framework: React.js

- **Backend:**
  - Node.js con Express.js

- **Base de Datos:**
  - PostgreSQL

- **Autenticación y Autorización:**
  - Auth0

- **Procesamiento de Pagos:**
  - Mercado pago

- **Almacenamiento de Archivos:**
  - Trailway para Base de datos y back, Vercel para front
